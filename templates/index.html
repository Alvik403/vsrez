Консолидация Excel-файлов

Шаблон теперь можно хранить внутри проекта. Приложение берёт его из конфига по умолчанию, а загрузка файла шаблона нужна только если вы хотите временно переопределить базовый файл.

Конфиг и логика

Базовый конфиг лежит в {{ default_config_path }}. Сейчас логика заточена под лист Справка по резервам и дописывает строки в блоки ЦФО, сохраняя формулы и оформление.

Если шаблон не загружен вручную, сервис возьмёт файл из default_template_path. Для новых блоков ЦФО используется копия существующего оформления с новым цветом заливки.

Схема JSON

default_template_path - встроенный путь до шаблона.

source_cols_preset - пресеты колонок в исходных файлах.

pages - правила заполнения листа и цветов новых блоков.

{
  "default_template_path": "Файлы/Шаблон ВСР.xlsm",
  "pages": [
    {
      "mode": "reserves_sheet_v1",
      "template_sheet": "Справка по резервам ",
      "source_sheet": "Лист1"
    }
  ]
}
